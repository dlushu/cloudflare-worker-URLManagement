# 🔗 URL 管理与短链接使用说明

这是一个简单的短链接工具，你可以：

* 添加短链接
* 访问短链接自动跳转
* 在网页里管理和删除链接

---

## 📌 一、如何登录

打开首页：

```
https://你的域名/
```

输入管理员密码登录即可。

---

## 📌 二、如何添加短链接（最重要）

使用接口添加（Webhook）

### 请求方式：

```
POST /webhook
```

### 示例（推荐用这个）：

```bash
curl -X POST https://你的域名/webhook \
  -H "Content-Type: application/json" \
  -d '{"name":"test","url":"https://example.com"}'
```

---

### 参数说明：

| 参数   | 说明    |
| ---- | ----- |
| name | 短链接名称 |
| url  | 跳转地址  |

---

### 示例结果：

```
https://你的域名/test
```

访问后会跳转到：

```
https://example.com
```

---

## 📌 三、如何使用短链接

直接访问：

```
https://你的域名/短链名称
```

例如：

```
https://你的域名/google
```

---

### ✅ 支持路径拼接

例如你设置：

```
name: github
url: https://github.com
```

访问：

```
https://你的域名/github/user/repo
```

会跳转：

```
https://github.com/user/repo
```

---

### ✅ 支持参数

```
https://你的域名/google?q=test
```

会自动带参数跳转。

---

## 📌 四、如何删除短链接

1. 打开管理页面
2. 点击对应卡片的 **删除按钮**

（需要登录状态）

---

## 📌 五、如何查看所有链接

打开：

```
https://你的域名/
```

登录后会显示：

* 所有短链接
* 原始地址
* 复制按钮
* 打开按钮
* 删除按钮

---

## 📌 六、返回说明（Webhook）

添加接口返回：

```json
{
  "success": true,
  "isUpdate": false
}
```

| 字段       | 说明        |
| -------- | --------- |
| success  | 是否成功      |
| isUpdate | 是否为更新已有链接 |

---

## 📌 七、常见问题

### ❓ 打不开短链

说明这个短链不存在，会返回：

```
404 Not Found
```

---

### ❓ 覆盖问题

同一个 `name` 会直接覆盖原来的链接。
